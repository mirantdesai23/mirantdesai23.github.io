# Mirant Desai

Personal website and blog built with Next.js 15 and deployed on GitHub Pages.

## About

This is the personal website of Mirant Desai, Founder & CEO of [PlanAI](https://www.planaipro.com).

## Tech Stack

- **Framework**: Next.js 15
- **Styling**: Tailwind CSS
- **Content**: MDX for blog posts
- **Deployment**: GitHub Pages with GitHub Actions

## Features

- 📝 Technical blog with MDX support
- 🎨 Dark theme matching PlanAI branding
- 📱 Fully responsive design
- ⚡ Static site generation for fast loading
- 🚀 Automated deployment with GitHub Actions

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
